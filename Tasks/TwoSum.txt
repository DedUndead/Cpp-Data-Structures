Find a pair of numbers N, K in sorted array, such that N + K = X
If such pair doesn't exist, output -1

Input: Array of N random sorted integers (any, non-negative), integer X (any, but non-negative)
Output: Pair of numbers

Example:
Input: [1 2 3 3 4 9] 6
Output: [2 4]

Input: [0 0 0 3 103 1002] 8
Output: -1
